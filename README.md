# snake
<h1>This is a Snake game for linux</h1>
<h1>This program only works on Linux PCs</h1>

<h2>Installation:</h2>
<ul>
<li>1. Download files:</li>
<li>
    <ul>
        <li>feld.c</li>
        <li>feld.h</li>
        <li>input.c</li>
        <li>input.h</li>
        <li>linkedlistlib.c</li>
        <li>linkedlistlib.h</li>
        <li>main.c</li>
        <li>output.c</li>
        <li>output.h</li>
        <li>snake.c</li>
        <li>snake.h</li>
        <li>timing.c</li>
        <li>timing.h</li>
    </ul>
</li>
<li>2. Compile it with cc main.c feld.c input.c linkedlistlib.c output.c snake.c timing.c -o snake</li>
<li>3. Copy game file to /bin directory with cp snake /bin/snake</li>
</ul>
<h2>How to run</h2>
<ul>
<li>You should do it from root user!</li>
<li>sudo snake</li>
</ul>
<h2>Control:</h2>
<ul>
<li>W - up</li>
<li>S - down</li>
<li>D - rigth</li>
<li>A - left</li>
</ul>
